Personalized Networking Assistant

Project Description

Personalized Networking Assistant is a web application that helps users prepare for networking events by generating personalized conversation starters based on event details and user interests. The application analyzes event information, suggests discussion topics, performs fact checking, and stores conversation history.

Technologies Used

2. FastAPI

3. Streamlit
